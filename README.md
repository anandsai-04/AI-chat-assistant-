# AI Engineering Assistant

A production-quality local AI Engineering Assistant built with FastAPI, Ollama, Qwen2.5-Coder, Sentence Transformers, and FAISS.

## Tech Stack
- **Python**: Core Language
- **uv**: Package Management & Virtual Environments
- **FastAPI**: Web Framework
- **Ollama & Qwen2.5-Coder**: Local LLMs
- **Sentence Transformers & FAISS**: RAG and Embeddings

## Setup
```bash
# Clone the repository
git clone <your-repo-url>
cd ai-assistant

# Sync dependencies and create a virtual environment
uv sync

# Activate the virtual environment
source .venv/bin/activate
```
